 🌟 Peshawar Todo

A simple and clean Todo List app built while learning Full Stack Development with the **Meta Professional Certificate** on Coursera.

 Features
- Add new tasks
- Mark tasks as done
- Delete tasks
- Filter: All / Active / Completed
- Tasks saved with localStorage (persist after refresh)
- Responsive design

 How to Run
1. Open the `web-version` folder
2. Open `index.html` in browser (Best with Live Server)

 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
